문제

사용자가 입력한 두 이름이 같은 성인지 다른 성인지 판단하는 문제입니다 이를 해결하는 다음 프로그램의 실행상태를 확인하세요 
A양, 이름을 입력하세요: 허난설헌
B군, 이름을 입력하세요: 허균
허난설헌과 허균은 성이 같습니다.

프로그램 코드

	 	#include <stdio.h>
 	#include <string.h> 				
	
1	int main() {																																				
		char nameA[10] = { '\0' };					
2		char nameB[10] = { '\0' };					
		
		printf( "A양, 이름을 입력하세요: " );
3		scanf( "%s", nameA );
		printf( "B군, 이름을 입력하세요: " );
4		scanf( "%s", nameB );
	
5		if ( !strncmp( nameA, nameB, 2 ) ) 
		{																																			
6			printf( "%s과 %s은 성이 같습니다. ", nameA, nameB );
		}																																			
7		else
		{																																			
8			printf( "%s과 %s은 성이 다릅니다. ", nameA, nameB );
		}																																			
		
9		return 0;
	}
