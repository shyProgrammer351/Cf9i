문제

기상 특보를 입력하면, 이를 바탕으로 주의사항을 출력하는 문제입니다 실행순서를 클릭하세요 참고로, 대설특보는 눈길 운전 조심을 출력하고, 호우특보는 빗길 운전 조심을 출력하고, 강풍주의보는 안전 운전 조심을 출력하세요.
오늘의 기상 특보를 입력하세요 : 대설특보
일부 지방에 대설특보가 발효중입니다. 출근길 눈길 운전에 유의해야 합니다.

프로그램 코드

	#include <stdio.h>

	#include <string.h>


1	int main() {

		char weatherReport[32] = "";

2		char newsMessage[1024] = "";

							

		printf("오늘의 기상 특보를 입력하세요 : " );

3		scanf( "%s", weatherReport );


		printf( "일부 지방에 %s가 발효중입니다. \n", weatherReport );

4		strcpy( newsMessage, "출근길 " );


5		if ( !strcmp( weatherReport, "대설특보" ) ) {

6			strcat( newsMessage, "눈길 " );

		}

7		else if ( !strcmp( weatherReport, "호우특보" ) ) {

8			strcat( newsMessage, "빗길 " );

		}

9		else if ( !strcmp( weatherReport, "강풍주의보" ) ) {

10			strcat( newsMessage, "안전 " );

		}

		

11		strcat( newsMessage, "운전에 유의해야 합니다." );


12		printf( "%s", newsMessage );


13		return 0;

	}
